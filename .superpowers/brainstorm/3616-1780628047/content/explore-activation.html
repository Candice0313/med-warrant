Phase 6 — How is explore mode activated?

The TierSelector lets you re-ground a verdict to any evidence tier and watch the confidence bar move live. The question is: how does that selector appear?

A — Always-on inline dropdown

Tier dropdown always visible on every row — no mode to activate. The current (real) tier is pre-selected. Changing it immediately re-caps and shows a visual "explored" indicator. Simplest interaction model.

B — Per-row "Explore" button

Each row has a small "Explore ↓" button that reveals a dropdown for that row only. Explored rows are visually transformed (different border, "↩ reset" link). Other rows stay untouched.

C — Global "Explore mode" toggle

A single toggle in the case header reveals tier dropdowns on all rows at once. Flipping it off hides all dropdowns and resets any changed tiers to their real values. Cleanest separation between "judged view" and "what-if view."
