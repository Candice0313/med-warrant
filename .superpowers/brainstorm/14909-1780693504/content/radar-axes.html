Profile tab — what does each axis show?

The radar has one axis per dimension (4 for the warfarin case). What value is plotted?

A — Dual series: model + capped (recommended)

Outer dashed = model confidence. Inner solid = capped/displayed. Mirrors the dual bar in each DimensionRow. The gap between them shows how much evidence pulled confidence down.

B — Single series, colour-coded by band

One value per axis (capped confidence). Each axis line and dot coloured green / amber / red by its trust band. Simple to read at a glance.

C — Evidence tier strength per dimension

Axes represent evidence tier (T1 outer = strongest, T6 inner = model prior). Shows which dimensions are well-grounded vs. ungrounded, independent of confidence values.
